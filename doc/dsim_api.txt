DSIM QUICK INTRODUCTION

Main variable types used in this DSIM specification are:
	<void>   - self explanatory
	<string> - a string argument, like *"argument"*
	<int>	 - an integer argument, like *100*
	<float>	 - a float argument between 0 and 1, like *0.75*
	<time>	 - specifies an event time in seconds:milliseconds, like *100:00*
	<id>	 - an id assigned as a variable from a DSIM api call. There are several
			   types of ids, including:
			   <id-plugin>
			   <id-cdf>
			   <id-network>
			   <id-hostname>

			   You use id types as you would a normal variable. For example,
			   the load_plugin(<string> plugin_file_path) api call returns an id
			   of type <id-plugin>, so: 
					my_plugin = load_plugin(<string> plugin_file_path)
			   would store an id for the loaded plugin in my_plugin. my_plugin
			   can then be used in other api calls that take type <id-plugin> as an
			   argument.

specify event times using:
on(time <time>) {
    api_function_call();
	...
}

Use the api calls below. Each event will be executed at that time during the simulation.

DSIM API SPECIFICATION:

<id-plugin> <-- load_plugin(<string> plugin_file_path)

<id-cdf> <-- load_cdf(<string> cdf_file_path)

<id-cdf> <-- generate_cdf(<int> base_center, <int> base_width, <int> tail_width)

<id-network> <-- create_network(<id-cdf> intranet_latency, <float> intranet_reliability)

<void> <-- connect_networks(<id-network> network1, <id-cdf> internet_latency_1to2, <float> internet_reliability_1to2, <id-network> network2, <id-cdf> internet_latency_2to1, <float> internet_reliability_2to1)

<id-hostname> <-- create_hostname(<string> base_hostname)

# One of upstream_bandwidth or downstream_bandwidth <id-cdf> values may be replaced with NULL to specify symmetric upstream and downstream bandwidth for the nodes. The symmetric bandwidth will be pulled from the non-NULL cdf.
<void> <-- create_nodes(<int> quantity, <id-plugin> application_vm, <id-network> host_network, <id-hostname> base_hostname, <id-cdf> upstream_bandwidth, <id-cdf> downstream_bandwidth, <id-cdf> cpu_speed, <string> application_vm_arguments)

<void> <-- end(<void>)
